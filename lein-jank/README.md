# lein-jank

A Leiningen plugin to manage, build, and test Jank native projects.

## Prerequisites
Make sure you have the `jank` executable installed and available on your system's `PATH`.

## Setup

**Create a project:** `lein new org.jank-lang/jank hello_lein`
**Run the project:** `lein jank run`
**Run the tests:** `lein jank test`
**Compile the project:** `lein compile --output exe`

For more information run `lein jank help`
